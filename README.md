> [演示地址：https://blog.kuaidjun.com](https://blog.kuaidjun.com)

## 简介
> 快斗博客是一个纯个人开发维护的前端知识分享博客！

## 技术栈

### 前端
#### 1. 前端三大件HTML+CSS(Sass)+JS
#### 2. 框架选择Vue+Vue-router
#### 3. UI库选择Iview库(别问为什么，因为好看)
#### 3. 数据请求使用axios

### 后端
#### 1. 服务端语言采用Nodejs
#### 2. 框架选择Express
#### 3. 数据库采用Mysql+redis
